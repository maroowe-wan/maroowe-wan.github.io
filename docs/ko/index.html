Kafka 중급

Kafka 초급을 마친 학습자를 위한 운영·신뢰성·연동·스트림 처리 확장 강좌입니다. 복제(Replication)와 ISR로 고가용성·내구성을 설계하고, at-least-once/exactly-once 같은 전달 보장과 멱등 프로듀서를 이해합니다. 파티셔닝 전략과 커스텀 파티셔너로 순서 보장·부하 분산을 제어하고, Schema Registry와 Avro로 스키마 진화를 다룹니다. Kafka Connect로 코드 없이 외부 시스템을 연동하고, Kafka Streams로 상태 기반 스트림 처리를 시작하며, 보존·로그 압축과 운영·모니터링 기초까지 실무 역량을 한 단계 끌어올립니다.

Kafka 초급(메시징 필요성, Topic·Partition·Offset·Broker, 아키텍처/ZooKeeper vs KRaft, Producer 기초·acks·키, Consumer·Consumer Group, 로컬 설치 end-to-end 실습)을 수료한 학습자. 운영·신뢰성·연동·스트림 처리 역량을 넓히려는 중급 학습자 · 총 7강
