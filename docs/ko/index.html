Kafka 중급

Kafka 초급을 마친 학습자를 위한 운영·신뢰성·연동·스트림 처리 확장 강좌입니다. 복제(Replication)와 ISR로 고가용성·내구성을 설계하고, at-least-once/exactly-once 같은 전달 보장과 멱등 프로듀서를 이해합니다. 파티셔닝 전략과 커스텀 파티셔너로 순서 보장·부하 분산을 제어하고, Schema Registry와 Avro로 스키마 진화를 다룹니다. Kafka Connect로 코드 없이 외부 시스템을 연동하고, Kafka Streams로 상태 기반 스트림 처리를 시작하며, 보존·로그 압축과 운영·모니터링 기초까지 실무 역량을 한 단계 끌어올립니다.

Kafka 초급(메시징 필요성, Topic·Partition·Offset·Broker, 아키텍처/ZooKeeper vs KRaft, Producer 기초·acks·키, Consumer·Consumer Group, 로컬 설치 end-to-end 실습)을 수료한 학습자. 운영·신뢰성·연동·스트림 처리 역량을 넓히려는 중급 학습자 · 총 7강

쿠버네티스 고급

쿠버네티스 중급2까지 마친 학습자를 위한 최상위 심화 강좌입니다. CRD와 Operator로 쿠버네티스 API 자체를 확장하고, Admission Control로 정책을 강제합니다. VPA·Cluster Autoscaler·KEDA로 고급 오토스케일링을 구성하고, GitOps(ArgoCD)와 프로그레시브 딜리버리(Canary·Blue-Green)로 배포를 자동화하며, Istio 서비스 메시로 트래픽 관리·mTLS·관측성까지 다룹니다.

중급2까지 마친 학습자 (RBAC·스케줄링·NetworkPolicy·Helm·모니터링을 이미 아는 수준) · 총 7강

쿠버네티스 초급

컨테이너 오케스트레이션이 왜 필요한지부터 시작해, Pod·Deployment·Service·YAML 매니페스트까지 쿠버네티스의 핵심 개념과 실습을 체계적으로 다루는 입문 강좌입니다.

쿠버네티스를 처음 접하는 입문자/초급자 (컨테이너·도커 기초 정도만 아는 수준) · 총 5강

쿠버네티스 중급

쿠버네티스 기초를 익힌 학습자를 위한 실무 중심 강좌입니다. ConfigMap·Secret으로 설정 분리, 영구 볼륨(PV·PVC) 스토리지, 헬스체크와 무중단 배포, 리소스 관리와 오토스케일링, Ingress 트래픽 라우팅까지 한 단계 더 나아간 운영 역량을 다룹니다.

쿠버네티스 기초(Pod, Deployment, Service, YAML 매니페스트)를 이미 아는 학습자. 실무 운영·배포 안정화 단계로 한 단계 더 나아가려는 중급 학습자 · 총 5강

쿠버네티스 중급2

쿠버네티스 중급1을 마친 학습자를 위한 실무 운영 확장 강좌입니다. StatefulSet으로 상태있는 워크로드를 다루고, Job·CronJob·DaemonSet 같은 다양한 워크로드 컨트롤러를 익힙니다. Namespace 멀티테넌시와 ResourceQuota, RBAC 권한 제어, 스케줄링 제어(Affinity·Taints), NetworkPolicy 트래픽 통제, Helm 패키징, 그리고 Prometheus·Grafana 모니터링과 로깅까지 클러스터 운영 역량을 한 단계 더 끌어올립니다.

쿠버네티스 중급1(ConfigMap/Secret, PV/PVC, Probes·롤링업데이트·롤백, requests/limits·HPA, Ingress)을 마친 학습자. 실무 운영 역량을 더 넓히려는 중급 학습자 · 총 8강
