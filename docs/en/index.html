Introduction to DevOps

A beginner course for developers and operations engineers encountering DevOps for the first time. Starting from the friction between development and operations teams, it covers DevOps culture and principles (CALMS), Git-based collaboration, CI/CD, Infrastructure as Code, and monitoring/observability — combining core concepts with foundational hands-on practice at every step.

Complete beginners to DevOps — developers or operations engineers who have heard terms like containers and CI/CD but have never studied them systematically. · 8 lectures

CI/CD Pipeline in Practice — Automated Deployment to AWS EC2 with GitLab and Jenkins

A hands-on course where you build a complete CI/CD pipeline from scratch: code pushed to GitLab is automatically built, tested, containerized, and deployed to an AWS EC2 server via Jenkins and docker compose. A simple, language-agnostic web app serves as the example, but the core skills — writing a Dockerfile, a Jenkinsfile, and the deployment logic — transfer directly to your own projects.

Developers setting up a CI/CD pipeline for the first time (comfortable with Git and the terminal; want to automate deployments to a server like EC2) · 8 lectures

CI/CD Pipeline in Practice — Automating AWS EKS Deployments with GitLab + Jenkins

A hands-on course for developers who want to deploy to Kubernetes (AWS EKS). Rather than diving deep into Kubernetes internals, the focus is on building a GitLab + Jenkins pipeline that automatically deploys your code to EKS. You will set up everything from image build and push, to writing deployment manifests, to kubectl rollouts and rollbacks — constructing a declarative deployment pipeline from scratch.

Developers who want to deploy to Kubernetes (EKS). Docker basics required; light prior exposure to Kubernetes is sufficient — deep internals knowledge is not needed and can be supplemented with dedicated Kubernetes courses. · 8 lectures

Kafka for Beginners

Starting from the fundamental question of why messaging and event streaming are needed, this beginner course covers core concepts — Topic, Partition, Offset, and Broker — along with Kafka's architecture, and finishes with a hands-on end-to-end exercise where you produce and consume messages directly using a Producer and Consumer.

Beginners and newcomers to Kafka who are familiar with basic server/networking concepts and simple CLI usage. Advanced topics such as replication, delivery guarantees, Kafka Connect, Streams, and operations are covered in the follow-up 'Kafka Intermediate' track. · 6 lectures

Kafka Intermediate

A continuation course for learners who have completed Kafka Basics, covering operations, reliability, integrations, and stream processing. You will design high availability and durability using replication and ISR, understand delivery guarantees such as at-least-once and exactly-once along with idempotent producers, control ordering guarantees and load distribution through partitioning strategies and custom partitioners, manage schema evolution with Schema Registry and Avro, integrate external systems without code using Kafka Connect, get started with stateful stream processing via Kafka Streams, and round out your practical skills with retention, log compaction, and essential operations and monitoring.

Learners who have completed Kafka Basics (messaging fundamentals, Topic/Partition/Offset/Broker, architecture/ZooKeeper vs KRaft, Producer basics/acks/keys, Consumer/Consumer Groups, local end-to-end lab). Intermediate learners looking to expand their skills in operations, reliability, integrations, and stream processing. · 7 lectures

Learners who have completed Kubernetes Intermediate 2 (already familiar with RBAC, scheduling, NetworkPolicy, Helm, and monitoring) · 7 lectures
